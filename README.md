# 🧠 Alzheimer’s Disease Prediction (Multimodal AI)

A smart AI-based system designed to **predict Alzheimer’s Disease** using **multimodal data (text, clinical data, and cognitive inputs)** — built without relying on medical imaging.

---

## 📌 Project Overview

This project leverages **Machine Learning and AI techniques** to analyze patient data and predict the likelihood of Alzheimer’s disease at early stages.

Unlike traditional systems, this model works **without MRI/CT images**, making it more accessible and cost-effective.

---

## 🧠 Features

* ✅ Multimodal input (patient data, symptoms, cognitive tests)
* ✅ Early-stage Alzheimer’s prediction
* ✅ No medical imaging required
* ✅ Fast and lightweight model
* ✅ User-friendly interface
* ✅ Real-time prediction

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/alzheimer-prediction.git
cd alzheimer-prediction
```

---

### 2️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

## 📊 Input Parameters

The model uses the following inputs:

* Age
* Gender
* Memory test scores
* Cognitive ability score
* Behavioral symptoms
* Medical history

---

## 📡 API Example

### Request

```json
{
  "age": 65,
  "memory_score": 45,
  "cognitive_score": 50
}
```

### Response

```json
{
  "prediction": "High Risk of Alzheimer’s"
}
```
